# 🐰 RabbitMQ Messaging Example

This project demonstrates how to use **RabbitMQ** to send and receive messages between services.

RabbitMQ is a **message broker** that allows applications to communicate asynchronously using **queues**.

It is widely used in **microservices**, **background jobs**, and **distributed systems**.

---

# 📌 What is RabbitMQ?

RabbitMQ is a **message broker** that enables applications to send messages to each other through **queues**.

Instead of services communicating directly, they exchange messages through RabbitMQ.

This improves:

- scalability
- reliability
- fault tolerance
- asynchronous processing
